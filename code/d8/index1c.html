vòng lặp for và vòng lặp while() thì 2 vòng lặp chỉ chạy khi mà điều kiện 
        của vòng lặp là true 
        for (statement1; điều kiện; statement3) {
            // code thực thi của vòng lặp 
        }
        vòng lặp while 
        while(điều kiện){
            // code thực thi của vòng lặp
        }

        => vòng lặp do ... while (điều kiện) 
        khác ở chỗ lần chạy đầu tiên nó không kiểm tra điều kiện 
        và nó chỉ kiểm tra điều tra điều kiện từ lần chạy thứ 2
