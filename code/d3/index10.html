thuộc tính flex-flow của flex container
        flex-flow là thuộc tính viết tắt của cả 2 thuộc tính flex-direction và flex-wrap

        .intro {
            display: flex;
            flex-direction: column;
            flex-wrap: wrap;
        }

        viết rút gọn lại 
        .intro {
            display: flex;
            flex-flow:column wrap; 
        }

        cú pháp :
        flex-flow: A B;
        trong đó A là giá trị của flex-direction
        B là giá trị của flex-wrap
